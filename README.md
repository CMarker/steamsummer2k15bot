steamsummer2k15bot
---------------------
This is a script to automate the Steam Summer Sale 2015 Monster game from your browser.

To use this script, click and drag this link >>>> <a href="javascript:(function(){document.body.appendChild(document.createElement('script')).src='https://rawgit.com/CMarker/steamsummer2k15bot/master/bot.js';})();">Start Steam Bot</a> <<<< to your bookmarks, start playing the game then click the bookmark.
